# odds-gap-analyzer-for-horse-race
馬連と馬単のオッズ差から市場の「歪み」を検知し、期待値の高い買い目を抽出するデータ分析ツールです。

## 📌 プロジェクト概要
自身がよく行なっている馬連での投票において、予想の精度を高めるために「馬連と馬単のオッズの乖離」に着目しました。
本ツールは、その乖離を独自の指標 **ΔP（人気順位の差）** として数値化し、過小評価されている（＝期待値が高い可能性がある）買い目を自動的に抽出します。

## 📊 分析とロジック
1. **データの正規化**: 馬連と馬単の組番の名寄せを行い、比較可能な状態に加工。
2. **市場乖離の計算**: `馬連人気順 - 馬単人気順` を算出し、乖離の大きさを可視化。
3. **期待値の算出**: 歪みが大きい順にソートし、効率的な投資判断を促進。

## 🛠 使用技術・使用ツール
- **Python 3.x**
- **Pandas** (データ加工・分析)
- **Jupyter Notebook** (.ipynb)
- **Excel** (データソース)

## 📁 使い方
1. `sample_data.xlsx`（オッズデータ）を用意します。
2. ノートブックを実行すると、歪みの上位30件が表形式で出力されます。
